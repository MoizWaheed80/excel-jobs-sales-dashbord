# 📊 Jobs Salary Dashboard (Microsoft Excel)

![excel-jobs-sales-dashbord](images/Excel Dashbord.png)

## Overview

The **Jobs Salary Dashboard** is an interactive Excel dashboard that provides a comprehensive analysis of job market data across different countries. It enables users to explore salary trends, job demand, employment types, and job categories through dynamic visualizations and KPI cards.

All reports are **fully dynamic**, meaning every chart, KPI, and visualization updates instantly based on the selected filters.

---

## Dashboard Features

### 🎯 Interactive Slicers

The dashboard includes three slicers at the top:

* **Job Category** – Filter data by a specific job title (e.g., Business Analyst, Data Analyst, Software Engineer).
* **Country** – View statistics for any available country.
* **Job Type** – Filter by employment type (Full-time, Part-time, Contractor, Temporary).

Every slicer is connected to all PivotTables and charts, allowing users to analyze data from different perspectives instantly.

---

## 📈 KPI Cards

The dashboard displays key performance indicators, including:

* **Total Job Count**
* **Minimum Salary**
* **Maximum Salary**

These KPIs automatically recalculate based on the selected slicer values.

---

## 📊 Visualizations

### Job Category Analysis

Displays the average salary across different job categories, allowing users to compare earning potential between roles.

### Job Type Analysis

Compares salaries across employment types such as:

* Full-time
* Part-time
* Contractor
* Temporary

### Global Salary Distribution

A world map visualizes salary distribution across countries, making it easy to identify regional salary trends.

---

## Dynamic Functionality

The dashboard is completely dynamic:

* Selecting a different **country** updates every chart and KPI.
* Changing the **job category** refreshes all metrics instantly.
* Filtering by **job type** automatically recalculates all statistics.
* All PivotTables, PivotCharts, and calculations remain synchronized.

## Dashboard Security

To protect the dashboard and ensure a consistent user experience:

* ✅ Dashboard sheet is protected to prevent accidental edits.
* ✅ Charts, KPI cards, and visual elements are locked.
* ✅ Users can interact only with slicers and filters.
* ✅ Transformation and helper sheets are hidden to keep the workbook clean and secure.
* ✅ Only the final dashboard is exposed to end users.
